Need Support?
=============

There are various ways to get support for the Solr extension:

Community Support
-----------------

Slack Channel
^^^^^^^^^^^^^

A dedicated channel on slack can be be used to get in touch with other users! Slack is a realtime team collaboration tool. There are a web app and native apps for Mac, Windows, iOS and Android.

The URL is: https://typo3.slack.com/messages/ext-solr/

.. note::

	If you are not yet registered, use http://forger.typo3.org/slack for that!

Mailinglist
^^^^^^^^^^^

If you need help with the news extension, you can always ask in the public mailinglist *typo3.projects.solr*.

An archive and the possibility to subscribe to the list can be found here: http://lists.typo3.org/cgi-bin/mailman/listinfo/typo3-project-solr

Paid Support
------------

Paid support is available by the Apache Solr team at dkd. They offer to help you setup your Apache Solr Enviroment. If any customization is required, they will fulfill your wishes.
